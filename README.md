# wanandroid-compose

Jetpack Compose 版本的「玩 Android」

## 环境
- AndroidStudio: Arctic Fox | 2020.3.1

## 功能
- 登录、注册
- 等级、排名、积分、收藏
- 首页、搜索
- 广场
- 公众号
