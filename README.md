# wanandroid-compose

Jetpack Compose 版本的「玩 Android」

## 环境

- AndroidStudio: 

![](https://raw.githubusercontent.com/wangchenyan/wanandroid-compose/master/art/android-studio.jpg)

- Gradle: 7.0.0-beta02
- Kotlin: 1.4.30
- Compose: 1.0.0-beta07

## 功能
- 登录、注册
- 等级、排名、积分、收藏
- 首页、搜索
- 广场
- 公众号
